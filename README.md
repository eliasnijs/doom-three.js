# MiniDoom
By Elias Nijs & René Van Der Schueren

🎮 Play it [here](https://eliasnijs.com/projects/minidoom/index.html). 🎮

### Installation
```
git clone git@github.com:eliasnijs/doom-three.js.git
cd doom-three.js
npm i
```

### Development
```
npm run dev
```
